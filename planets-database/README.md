# Planets Database

This repository is a practice project for learning Git and GitHub workflows.
It contains text files with basic facts about planets in our solar system and
a CSV file with planet statistics.

## Contents
- inner-planets/: text files with facts about inner planets
- outer-planets/: text files with facts about outer planets
- data/planet-stats.csv: basic planet data in CSV format
- data/fun-facts.txt: interesting facts about planets

## Overview
The inner planets are rocky and close to the Sun.
The outer planets are much larger and mostly gas giants.

Fun fact: Venus is hotter than Mercury even though Mercury is closer to the Sun.
